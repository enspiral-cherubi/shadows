## shadows

a city and some lights, deployed here https://topological-ragnar.github.io/shadows

### install

```
git clone https://github.com/enspiral-cherubi/slush2
cd slush2
npm install
```

### develop

```
npm start
```

browse to <http://localhost:9966/>.

### test

```
npm test
```

### deploy

```
npm run deploy
```
